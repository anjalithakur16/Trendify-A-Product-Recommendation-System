**2. Short Description / Purpose**

Trendify is a data-driven fashion trend analysis and recommendation system designed to help businesses and users identify trending fashion products based on customer ratings, categories, and preferences.

The project analyzes fashion product data and presents actionable insights through an interactive dashboard, while also generating personalized recommendations .

**Purpose:**

Identify trending fashion products

Help businesses understand customer demand

Provide users with smart fashion recommendations

Support data-driven decision-making in retail
